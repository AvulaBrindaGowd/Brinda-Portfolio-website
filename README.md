# Brinda Gowd Avula — Personal Portfolio

A responsive, single-page personal portfolio website built with HTML, CSS, and JavaScript.

---

## 🌐 Live Preview

Open `brinda_portfolio.html` in your browser or run with **Live Server** in VS Code.

---

## 📁 Project Structure

```
portfolio/
├── brinda_portfolio.html   # Main portfolio file (HTML + CSS + JS)
├── BrindaAvula_Resume.pdf  # Resume file (linked to Download button)
└── README.md               # Project documentation
```

---

## ✨ Features

- **Single-page layout** with smooth scroll navigation
- **Fixed navbar** with active highlight on scroll
- **Hero section** with floating skill tags and avatar
- **About section** with stats cards
- **Skills section** grouped by category
- **Projects section** with tech chips
- **Education** timeline
- **Experience** timeline with responsibilities
- **Certifications** grid
- **Contact section** with social links
- **Dark theme** with purple/teal accent colors
- **Noise texture** background effect

## 🚀 How to Run

### Option 1 — Direct Browser
1. Double-click `brinda_portfolio.html`
2. Opens in your default browser

### Option 2 — Live Server (Recommended)
1. Open folder in **VS Code**
2. Install **Live Server** extension by Ritwick Dey
3. Right-click `brinda_portfolio.html` → **Open with Live Server**
4. Auto-reloads on every save

---

## 📄 Resume Download

Make sure `BrindaAvula_Resume.pdf` is in the **same folder** as `brinda_portfolio.html` for the Download Resume button to work.
